# HandGestureDetection
 using open cv for hand gesture detection
you can find here the project for the seconde task hand gesture recognition using openCV and instead of using a predefined input we used a mask and the camera to get the input live while the camera is recording and outputing it straightly to the view of the camera on the screen to the user
instruction
to use the app make sure that the light conditions are good and keep your hand inside the small box drawn
